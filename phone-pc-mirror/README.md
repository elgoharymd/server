# 📱 ScreenMirror Pro - عكس شاشة الهاتف على الكمبيوتر

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Node.js](https://img.shields.io/badge/Node.js-18.x-green.svg)](https://nodejs.org/)
[![Socket.io](https://img.shields.io/badge/Socket.io-4.x-black.svg)](https://socket.io/)

## 🌟 المميزات

- 🚀 **بث مباشر** عالي الجودة لشاشة الهاتف
- 🖱️ **تحكم كامل** بالفأرة من الكمبيوتر
- ⌨️ **دعم لوحة المفاتيح** الفعلية
- 📱 **رمز QR** للاتصال السريع
- 🔒 **نظام جلسات** آمن
- ⚡ **ضغط ذكي** للصور لتقليل استهلاك البيانات
- 📊 **إحصائيات** لحظية للأداء
- 🎨 **واجهة حديثة** وجذابة

## 🛠️ التقنيات المستخدمة

- Node.js + Express
- Socket.IO للاتصال الفوري
- Sharp لضغط الصور
- QR Code للاتصال السريع

## 📦 التثبيت

```bash
# clone المشروع
git clone https://github.com/yourusername/phone-pc-screen-mirror.git
cd phone-pc-screen-mirror

# تثبيت الحزم
npm install

# نسخ ملف البيئة
cp .env.example .env

# تشغيل الخادم
npm start
